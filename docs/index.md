# IR-Based Stress Detection

Unsupervised rPPG extraction from near-infrared (NIR) in-cabin video using Contrast-Phys+, with heart-rate variability stress indicators (Baevsky SI, SDNN, RMSSD, pNN50).

**Repository:** [github.com/neha191091/IR-based-stress-detection](https://github.com/neha191091/IR-based-stress-detection)
